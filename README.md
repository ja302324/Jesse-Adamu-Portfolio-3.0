# Jesse Adamu Portfolio

A personal portfolio site for Jesse Adamu, built to showcase graphic design, visual storytelling, branding work, and motion-driven digital experiences.

This project is built with React and Vite, with animated section transitions, a moving image-grid hero, and a horizontally scrolling selected work section.

## Overview

The site is structured around a few core sections:

- A visual hero section with animated artwork using `GridMotion`
- A selected work section with horizontally scrolling project cards
- An about section introducing Jesse's design perspective
- A contact section for outreach and collaboration

## Tech Stack

- React 19
- Vite 7
- Motion
- GSAP
- Three.js
- Tailwind CSS 4
- ESLint

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Design Direction

This portfolio leans into cinematic visuals, bold graphic presentation, and motion as part of the storytelling. The goal is to present creative work in a way that feels intentional, immersive, and modern.
